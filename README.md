# Landing-Page
Nesse projeto acadêmico nós (alunos), tivemos o desafio de criar um projeto que fosse capaz de: receber dados como nome, foto de perfil, email e telefone e também mudar a cor do tema (theme), e ainda salvar esses dados para quando o usuário retornar com o mesmo user (nome de usuário) os dados estejam salvos!
